# skill-spark

Universal skill manager for AI coding agents.

## Installation

```bash
bun install
```

## Commands

### search [query]
Search skills from registry or browse interactively.

### add <source>
Install skills from Git URLs, well-known endpoints, or flins directory.

### list
List installed skills and commands.

### outdated [skills...]
Check for updates and missing files.

### update [skills...]
Update installed skills to latest versions.

### remove [skills...]
Remove installed skills.

### map --target <agent>
Map skills to target agent directory.

### doctor
Diagnose environment and detect installed agents.

## Supported Agents

42+ AI coding agents including Claude Code, Cursor, Codex, Gemini CLI, and more.
