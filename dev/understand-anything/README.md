# Understand Anything Skill

This skill provides integration with the [Understand Anything](https://github.com/Lum1104/Understand-Anything) plugin for AI coding assistants.

## Quick Start

1. **Install this skill** to your project or system skills directory
2. **Install the plugin** (for supported platforms):
   - Claude Code: `/plugin marketplace add Lum1104/Understand-Anything && /plugin install understand-anything`
   - Trae: `/plugin marketplace add Lum1104/Understand-Anything && /plugin install understand-anything`
3. **Use the commands**: `/understand`, `/understand-dashboard`, etc.
