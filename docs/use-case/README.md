## 搜索命令

```bash
# JSON 格式
dist/skill-spark search "关键词" --output results.json

# Markdown 格式
dist/skill-spark search "关键词" --output results.md --format markdown

# 自动检测格式 (.md 扩展名自动用 markdown)
dist/skill-spark search "关键词" --output results.md
```
