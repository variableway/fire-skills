# Phase 0: 内容收集策略

> 输入一个名字 → 从零搭建调研素材库。此文件仅在 agent 不确定信源优先级或搜索策略时加载。

## 搜索顺序（必须遵守）

不是搜一堆结果然后挑。顺序是强制的——每一层确认质量后再进下一层：

```
著作与长文 → 长对话/播客 → 社交媒体/碎片 → 外部批评 → 行为证据 → 时间线
     ↑              ↑               ↑               ↑            ↑          ↑
  精心编辑的      即兴反应        碎片人格       自己的盲区    嘴上vs手上    演化轨迹
```

为什么这个顺序：
- 著作给了你此人的**理想形象**和**词汇系统**——没有这个基础，你看播客时会找不到关键概念
- 播客暴露**即兴状态下的真实反应**——和著作的差异本身就是数据
- 社交媒体暴露**本能**——15 字的推文比 200 页的书更诚实
- **批评必须在消化完一手材料之后再读**——否则你会被批评者的框架绑定，看不到批评本身的盲区

## 每层搜索的具体方法

### 层 1: 著作与系统思考

```
搜索模板: "{name} book" "{name} essay" "{name} blog" "{name} tweetstorm"
优先源: 此人的个人网站、Medium、Substack、nav.al 风格的个人博客
```

收集目标：
- 此人最著名的 5-10 篇文章/章节
- 此人反复使用的概念/术语/框架
- 此人的写作风格特征（句式、修辞、引用习惯）

### 层 2: 长对话与即兴思考

```
搜索模板: "{name} podcast" "{name} interview" "{name} Joe Rogan / Tim Ferriss / Lex Fridman"
优先源: 有完整 transcript 的播客（比摘要/评论可靠 10 倍）
```

收集目标：
- 此人在被追问时的反应模式（拆解定义？反问？承认不知道？回避？）
- 此人和著作中不一致的表述
- 此人用过的即兴类比和比喻

### 层 3: 碎片表达与实时反应

```
搜索模板: "{name} Twitter / X" "{name} quote" "{name} social media"
优先源: Twitter/X（如果有），Reddit AMA（如果有）
```

收集目标：
- 15-50 条原始推文/帖子（不是别人转述的）
- 此人在社交媒体上争论/被反驳时的反应
- 高频词汇、表情使用、发帖节奏

### 层 4: 外部批评与对立观点

```
搜索模板: "{name} criticism" "{name} controversy" "{name} Hacker News"
         "{name} Reddit" "{name} Goodreads review" "{name} critique"
警告: 不要在读完层 1-3 之前读这层！
```

收集目标：
- 学术界对方法的批评
- 社区对比人的合理性质疑
- 此人的逻辑盲区和利益冲突（crypto 上的金融利益、硅谷特权视角等）

### 层 5: 决策记录与行为证据

```
搜索模板: "{name} investment history" "{name} career decisions"
         "{name} company founding / selling / pivoting"
```

收集目标：
- 此人的关键投资/商业决策时间线
- 每次决策的上下文（为什么选 A 不选 B）
- 此人说什么 vs 做什么的不一致

### 层 6: 时间线与传记脉络

```
搜索模板: "{name} biography" "{name} childhood" "{name} early career"
         "{name} education" "{name} turning point"
```

收集目标：
- 出生地、家庭背景、教育 → 早期影响
- 关键转折事件（通常是被骗/被解雇/遭遇失败）
- 此人在不同年龄段的核心关注点演变

## 信源质量分级

| 等级 | 标准 | 示例 |
|------|------|------|
| ⭐⭐⭐⭐ | 一手信息，本人产出 | 推文、播客、博客、公开演讲 |
| ⭐⭐⭐ | 二手信息，有引用的分析 | 学术文章、有引用的传记 |
| ⭐⭐ | 二手信息，有证据的评论 | 深度报道、采访转录 |
| ⭐ | 二手信息，无证据的观点 | 普通博客、社交媒体评论 |
| ❌ | 不消化 | 知乎、微信公众号、百度百科、AI生成的总结 |

## 每期素材要求

| 维度 | 最少素材数 | 自检标准 |
|------|-----------|---------|
| 著作 | 15+ | 包含此人的 3+ 个不同时期？ |
| 对话 | 15+ | 至少有 2 个不同采访者/主持人？ |
| 表达 | 15+ | 包含不同话题，不只是火的那条？ |
| 批评 | 15+ | 有学术批评和社区批评，不只有粉丝内容？ |
| 决策 | 10+ | 有时间跨度，不只是最近两年？ |
| 时间线 | 完整的 | 覆盖出生→当前？关键转折点不缺失？ |

## 常见失败模式

| 触发 | 修复 |
|------|------|
| 只搜到了此人的粉丝内容，没批评 | 搜 "{name} criticism Reddit" 或 "{name} Hacker News" |
| 搜到了一堆 GPT 生成的"名人名言"合集 | 加 "-名言 -金句" 排除，只留原始出处 |
| 所有素材都来自同一时期 | 在搜索词中加时间限定（"2010" "2020" "2024"） |
| 播客素材只有摘要没正文 | 搜 "{podcast name} transcript" 或 "{podcast name} episode {number} transcript" |
| 此人没有社交媒体 | 不做层 3，标注「此人无公开社交媒体足迹」 |
