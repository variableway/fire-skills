# 高保真设计

## 流程

1. 提问澄清需求
2. 找到现有 UI kit / 设计系统 / 参考代码 — 复制所有相关组件，读所有相关示例
3. 先产出假设 + 上下文 + 设计推理（含占位符），早点给用户看
4. 逐步构建设计，每次迭代都展示
5. 用工具验证、修复、迭代

**好的高保真设计不从零开始。** 花时间获取设计上下文：代码库、UI kit、截图、设计系统。从零虚构是最后手段。

## 美学方向

动手前，选定一个大胆的美学方向：
- **目的**：界面解决什么问题？谁在用？
- **调性**：极简 brutal / 极繁混沌 / 复古未来 / 有机自然 / 奢华精致 / 趣味玩具化 / 编辑杂志风 / brutalist 粗野 / Art Deco 几何 / 柔和粉彩 / 工业实用风
- **差异化**：什么让人过目不忘？

### 排版
选有特色的字体，避免 Arial/Inter 等通用字体。独特的展示字体 + 精致的正文字体搭配。

### 色彩
用 CSS 变量保持一致性。主色 + 强强调色，好过均匀分布的调色板。

### 动效
CSS-only 优先。一个精心编排的入场交错动画 > 散落的微交互。

### 空间
不对称、重叠、对角线流动、网格破坏、大量留白或受控密度。

### 氛围
渐变网格、噪点纹理、几何图案、分层透明、戏剧性阴影——不要默认纯色背景。

**每次生成都要变化**：明/暗主题、不同字体、不同美学。不要收敛到相同选择。

## 多方案展示

并排展示多个方案时：中性灰背景，每个选项放在自己的标签框里（小标题 + 白色圆角卡片），相关选项分组。

## 占位符

没有图标/素材时画占位符。高保真设计中，占位符好于乱画的假东西。

## 给选项

至少 3 个变体，跨多个维度。有的遵循现有模式，有的探索新颖交互。从基础开始逐步进阶。玩转品牌素材：缩放、填充、纹理、视觉节奏、层叠、新颖布局、字体处理。
