# 手工测试指南

> 如何验证 `thought-distiller` skill 是否能正常工作

## 前置条件

```bash
skill-spark add notable-figures/thought-distiller --agent trae --yes
```

## 单人蒸馏模式（只模仿一个人）

thought-distiller 支持两种模式：

### 模式 A：完整蒸馏（全流程）

```
请帮我蒸馏 Naval Ravikant 的思维框架

→ Agent 自动执行 Phase 0 → Phase 5 全流程
→ 产出完整 skill 目录（SKILL.md + references/ + examples/）
```

适用于：要创建一个可以独立安装和运行的人???Skill。

### 模式 B：快速咨询（不产出文件）

```
帮我用 thought-distiller 的框架分析一下 Naval Ravikant 的思维方式

→ Agent 只执行 Phase 2-3（交叉验证+结构化解码），不执行 Phase 0/1/5
→ 输出心智模型概要 + 决策启发式 + 表达DNA，不创建文件
→ 适用于：只是想理解一个人的思维框架，不需要产出可安装的 Skill
```

### 模式 C：仅调研不蒸馏

```
对 Naval Ravikant 做 Phase 0-1 的深度调研，整理六维素材，不做蒸馏

→ Agent 执行 Phase 0-1
→ 输出 6 个 reference 文件
```

## 测试场景（12 条）

### 测试 1：Phase 0 信源质量

| 输入 | 验证 |
|------|------|
| `调研 Naval Ravikant 的著作材料，要收集至少15条素材` | (a) 无知乎/公众号等低质源 (b) 每条有来源URL (c) 标注了等级 |

### 测试 2：Phase 0 收集顺序

| 输入 | 验证 |
|------|------|
| `开始调研 Elon Musk 的思想体系` | (a) 先搜著作/长文 (b) 外部批评不在前几次搜索中出现 |

### 测试 3：Phase 2 三重验证

| 输入 | 验证 |
|------|------|
| `Naval 的「杠杆思维」能否被认定为心智模型？按三重验证法分析` | (a) 检查了著作 (b) 检查了播客 (c) 检查了决策 (d) 给出三层结论 |

### 测试 4：Phase 2 矛盾标注

| 输入 | 验证 |
|------|------|
| `Naval 说「远离政治」但 2024 年公开站队。蒸馏时怎么处理？` | (a) 没选一边忽略 (b) 没编调和解释 (c) 标注为内在张力 |

### 测试 5：Phase 2 排他性测试

| 输入 | 验证 |
|------|------|
| `「持续学习很重要」「投资要买好公司」「反脆弱」哪些是真正的??智模型？` | (a) 前两个被排除 (b) 反脆弱通过 (c) 每个有解释 |

### 测试 6：Phase 3 模型数量控制

| 输入 | 验证 |
|------|------|
| `生成 Naval 的 SKILL.md` | (a) 心智模型3-7个 (b) 每个有局限性 (c) 每个有至少2场景证据 |

### 测试 7：Phase 3 表达DNA量化

| 输入 | 验证 |
|------|------|
| `Naval 六维调研已完成。提取他的表达DNA` | (a) 有统计数字 (b) 抽样了素材 (c) 不凭感觉 |

### 测试 8：Phase 4 已知测试

| 输入 | 验证 |
|------|------|
| `已完成 Naval SKILL.md。运行已知测试：「应该去大公司还是创业？」对比 Skill 推断和 Naval 真实立场` | (a) 方向一致 (b) 标注了分歧点 (c) 有测试过程记录 |

### 测试 9：Phase 4.5 交叉比对

| 输入 | 验证 |
|------|------|
| `对比 Naval 和 芒格的核心概念差异` | (a) 覆盖概念重叠 (b) 覆盖框架共享 (c) 覆盖思想传承 (d) 覆盖矛盾对立 |

### 测试 10：Phase 5 Progressive Disclosure

| 输入 | 验证 |
|------|------|
| `最终交付 Naval SKILL.md` | (a) 行数≤150 (b) 深参考在 references/ (c) 有透明度协议 |

### 测试 11：透明度协议

| 输入 | 验证 |
|------|------|
| 用生成的 Naval Skill 连续问 3 个问题 | 所有回答结尾有 `—` 标注。任一条缺→失败 |

### 测试 12：单人蒸馏——快速咨询模式

| 输入 | 验证 |
|------|------|
| `帮我用 thought-distiller 的框架快速分析一下 Steve Jobs 的思维框架，不产出文件` | (a) 跳过了 Phase 0 调研收集 (b) 输出了心智模型概要 (c) 标注了「这是推断，基于公开信息」 |
