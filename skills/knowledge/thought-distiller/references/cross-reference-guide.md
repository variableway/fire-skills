# Phase 4.5: 交叉比对方法论

> 当蒸馏了多个人物后，对人物之间的思想关系进行比对——发现思想谱系、区分表面相似和本质差异。

## 何时触发

满足任一条件即可触发交叉比对：
- 同一批任务中蒸馏了 ≥2 个人物
- 用户问 "A 和 B 的思维方式有什么不同？"
- 正在写某人的心智模型时，发现和另一个已蒸馏的人物高度重叠

## 四个比对维度

### 维度 1: 概念重叠（Same Concept, Different Words）

两个人用不同词说同一件事。这种比对揭示的是**包装差异**——同样的思想，不同的表达系统。

```
问: Naval 的"特定知识"和 芒格的"能力圈"是同一件事吗？

分析:
  Naval "specific knowledge" = 不可教 + 不可外包 + 做起来像玩
  芒格 "circle of competence" = 知道你知道什么 + 不要越界

  重叠部分: 都强调先找到自己独特的优势域
  关键差异: Naval 强调主动性（去发现你的 SK），芒格强调防守性（不要出圈）
→ 表面相似，本质不同。Naval 是进攻版的芒格。
```

**分析方法**：
1. 找出两人的核心概念列表
2. 找出语义空间上有重叠的概念对
3. 对比每个对的定义边界 → 找出不一致的边界

### 维度 2: 框架共享（Same Framework, Different Source）

两个人从完全不同的领域得到了相似的心智模型。这种比对揭示的是**思想的底层普适性**。

```
问: 塔勒布的"反脆弱"和 Naval 的"痛苦→系统重构"有关系吗？

分析:
  塔勒布"反脆弱" = 系统从冲击中获益而非受损
  Naval"痛苦→系统重构" = 不修个案，重构产生问题的系统

  共享框架: 都用"系统冲击 → 系统升级"的模式解释进步
  来源差异: 塔勒布从概率论和交易中得出，Naval 从创业失败中得出
→ 独立发现 + 不同领域的佐证 → 这个思想比任意一方更可信
```

### 维度 3: 思想传承（Who Influenced Whom）

还原思想谱系——这个人的核心概念是从谁来的，又影响了谁。

```
Naval Ravikant 的智识谱系:
  受影响: Taleb（反脆弱+skin in the game）、佛教（欲望即痛苦）、
          Felic（投资哲学）、爱默生（自立）
  影响了: 硅谷创业者（AngelList + 推文）、
          花叔（女娲的产物）← 可以标注
```

**分析方法**：
- 算高频引用：此人在播客/推文中提到最多的人
- 追踪概念继承：此人的核心概念和别人的核心概念是否有直系血缘关系
- 标注间接影响：即使没直接引用，概念结构明显相似也标注

### 维度 4: 矛盾对立（Irreconcilable Differences）

两个人对同一件事有完全相反的结论。这种比对最有价值——它暴露了各自隐藏的前提假设。

```
问: Naval 和 芒格在"投资"上的分歧是什么？

  Naval: "Code and media are permissionless leverage. Invest in yourself."
  芒格: "The big money is not in the buying or selling, but in the waiting."

  表面矛盾: Naval 强调建资产，芒格强调耐心等待
  深层统一: 两人都反对短线交易——只是 Naval 的方案是"别交易，建公司"，
           芒格的方案是"别交易，长期持有好公司"
→ 看似矛盾，实则在"不要用时间换钱"上完全一致，路径不同源于时代不同
```

## 输出格式

交叉比对完成后，输出一个表格：

```
| 人物 A | 人物 B | 重叠/差异 | 本质相同？ | 关键区别 |
|--------|--------|---------|----------|---------|
| Naval: 特定知识 | 芒格: 能力圈 | 概念重叠 | 否 | 进攻 vs 防守 |
| Naval: 痛苦→系统 | 塔勒布: 反脆弱 | 框架共享 | 是 | 来源不同 |
| Naval: 代码杠杆 | 芒格: 复利 | 概念重叠 | 是 | 同类思想，不同时代 |
```

如果发现了思想谱系关系，额外输出：

```
思想谱系图:
  塔勒布 → Naval（继承反脆弱、skin in the game）
  佛教/斯多葛 → Naval（继承欲望管理）
  Naval → 花叔（女娲产物）
```

## 限制

- 交叉比对依赖于已知的调研材料。如果某人还没被蒸馏，对比结果可能不完整
- 不要为了找关系而找关系——没有实际连接的两个人不应该被强行比对
- 标注推断。区分"明确的思想继承"（此人说过受影响）和"推测的思想平行"（结构相似但未确认）
