# Skill Workspace  Ideas

当前目录期望变成一个Skill的workspace,主要用来：
第一阶段: 
1.  收集Skill, Skill Marketplace search, 下载
2.  安装，验证SKILL
3.  可以组合的形式安装SKILL在本地运行，或者安装到不同的项目中

第二阶段：
1.  有一个客户端的软件，扫描SKILL目录，在页面上展示使用
2.  可以在客户端中运行SKILL
3. 可以有一个Skill Lib 这种方式，可以给不同的AI Agent 使用，同时也有自己的AI Agent
4. 可以结合其他Workflow 工具比如Dify，n8n进行这种Skill的编排和测试
